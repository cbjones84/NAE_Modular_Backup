# NAE Utility Scripts

This directory contains utility scripts for running and managing NAE.

## Scripts

### Automation
- `start_nae_automation.py` - Start the NAE master scheduler
- `automated_donnie_scheduler.py` - Automated Donnie cycle scheduler

### Agent Execution
- `run_ralph_cycle.py` - Run Ralph's strategy generation cycle
- `run_ralph_max_quality.py` - Run Ralph with high-quality settings
- `run_donnie_cycle.py` - Run Donnie's strategy processing cycle
- `run_all.py` - Run all agents

### E*Trade Setup
- `setup_etrade_oauth.py` - Complete E*Trade OAuth setup
- `quick_setup_etrade_oauth.py` - Quick OAuth setup
- `complete_etrade_oauth.py` - Complete OAuth flow
- `get_etrade_auth_url.py` - Get authorization URL

### Checking Status
- `check_optimus_profits.py` - Check Optimus profits
- `check_optimus_profits_detailed.py` - Detailed profit check
- `check_strategy_flow.py` - Check strategy pipeline
- `check_etrade_sandbox_readiness.py` - Check E*Trade readiness

### Other
- `quick_setup_anthropic.py` - Anthropic API setup
