# NAE Documentation

This directory contains all NAE documentation organized by category.

## Structure

### Setup Documentation (`setup/`)
Setup guides and configuration instructions:
- API key setup (Anthropic, E*Trade)
- Environment configuration
- OAuth setup guides
- Initial setup summaries

### User Guides (`guides/`)
How-to guides and usage documentation:
- `USAGE_GUIDE.md` - Main usage guide
- `AGENT_AUTOMATION_GUIDE.md` - Agent automation guide
- `QUICK_START_GUIDE.md` - Quick start guide
- And more...

### Summaries (`summaries/`)
Implementation summaries and status reports:
- Feature implementation summaries
- Security updates
- Integration summaries
- Status reports

## Quick Links

- **Getting Started**: See `guides/USAGE_GUIDE.md` or `guides/QUICK_START_GUIDE.md`
